## 1. Подготовка файлов

### Структура проекта

```
symple site/
├── index.html          # Главная страница
├── css/
│   └── styles.css      # Стили
├── js/
│   └── script.js       # Таймер, форма, меню
└── GUIDE.md            # Этот гайд
```

### Перед публикацией — обязательно измените

| Что | Где | На что заменить |
|-----|-----|-----------------|
| Email для заявок | `js/script.js`, строка `orders@example.com` | Ваш реальный email |
| Название бренда | `index.html` | Ваше название (если нужно) |
| Ссылки на оферту и политику | `index.html`, footer | Реальные страницы |

### Локальный предпросмотр

1. Откройте папку проекта
2. Дважды кликните `index.html` — сайт откроется в браузере
3. Или запустите локальный сервер (опционально):

```bash
# Python 3
python -m http.server 8080

# Node.js (если установлен npx)
npx serve .
```

Откройте `http://localhost:8080`

## 2. Публикация сайта

### Вариант A: GitHub Pages (бесплатно)

**Подходит если:** есть аккаунт GitHub, нужен бесплатный хостинг.

1. Создайте репозиторий на [github.com](https://github.com) (например, `siteforge-landing`)
2. Загрузите все файлы проекта
3. Перейдите: **Settings → Pages**
4. Source: **Deploy from branch** → `main` → `/ (root)` → Save
5. Через 1–2 минуты сайт будет доступен по адресу:
   `https://ваш-логин.github.io/siteforge-landing/`

```bash
git init
git add .
git commit -m "Initial landing page"
git branch -M main
git remote add origin https://github.com/ВАШ-ЛОГИН/siteforge-landing.git
git push -u origin main
```

---

### Вариант B: Netlify (бесплатно, рекомендуется)

**Подходит если:** нужен быстрый деплой, SSL, формы.

1. Зарегистрируйтесь на [netlify.com](https://netlify.com)
2. Перетащите папку проекта в Netlify Drop, или:
   - **Add new site → Import from Git** → подключите GitHub
3. Build settings: оставьте пустыми (статический сайт)
4. Deploy — готово!
5. Сайт получит адрес вида `random-name.netlify.app`

**Бонус — формы без бэкенда:**
- Добавьте в `<form>` атрибут `netlify` и скрытое поле:

```html
<form name="order" method="POST" data-netlify="true" netlify-honeypot="bot-field">
  <input type="hidden" name="form-name" value="order">
  <p style="display:none"><input name="bot-field"></p>
  <!-- остальные поля -->
</form>
```

Заявки будут приходить в Netlify Dashboard → Forms.

---

### Вариант C: Vercel (бесплатно)

1. Зарегистрируйтесь на [vercel.com](https://vercel.com)
2. **Add New Project** → импортируйте репозиторий или загрузите папку
3. Framework Preset: **Other** (статика)
4. Deploy

---

### Вариант D: Beget / REG.RU (платный хостинг, .ru домен)

**Подходит если:** нужен .ru домен и российский хостинг.

#### Beget (~200₽/мес)

1. Зарегистрируйтесь на [beget.com](https://beget.com)
2. Создайте сайт в панели управления
3. Откройте **Файловый менеджер** → папка `public_html`
4. Загрузите `index.html`, папки `css/` и `js/`
5. Сайт доступен по домену, привязанному к хостингу

#### REG.RU

1. Купите домен `.ru` (~200–500₽/год) + хостинг
2. В панели: **Хостинг → Файловый менеджер**
3. Загрузите файлы в корневую директорию сайта
4. Привяжите домен к хостингу в разделе DNS

## 3. Подключение домена

### Для Netlify / Vercel / GitHub Pages

1. Купите домен (REG.RU, Beget, Namecheap)
2. В панели DNS добавьте записи:

| Тип | Имя | Значение |
|-----|-----|----------|
| A | @ | IP хостинга (указан в панели) |
| CNAME | www | ваш-сайт.netlify.app |

3. В Netlify: **Domain settings → Add custom domain** → введите домен
4. SSL-сертификат выпустится автоматически (Let's Encrypt)

### Рекомендуемые домены для этого проекта

- `siteforge.ru`
- `saitza1000.ru`
- `ai-sait.ru`

## 4. Внедрение AI-агента

Три варианта — от простого к продвинутому.

---

### Вариант A: Telegram-бот + Cursor AI (рекомендуется для старта)

**Схема:** Форма → Telegram → вы запускаете Cursor Agent → отправляете клиенту ссылку.

#### Шаг 1: Создайте Telegram-бота

1. Напишите [@BotFather](https://t.me/BotFather) → `/newbot`
2. Получите **API Token**
3. Узнайте свой Chat ID: напишите [@userinfobot](https://t.me/userinfobot)

#### Шаг 2: Подключите форму к Telegram через n8n или Make

**Через Make.com (бесплатный план):**

1. Создайте сценарий: **Webhook → Telegram Bot**
2. Webhook URL вставьте в форму или используйте Netlify Forms + Make trigger
3. Сообщение в Telegram:

```
🔥 Новая заявка!
Имя: {{name}}
Телефон: {{phone}}
Описание: {{description}}
```

#### Шаг 3: Обработка заказа в Cursor

1. Получили заявку в Telegram
2. Открываете Cursor → новый проект
3. Промпт для AI-агента:

```
Создай одностраничный сайт по ТЗ клиента:
[вставить описание из заявки]

Требования:
- Современный дизайн, тёмная тема
- Адаптив под мобильные
- index.html + css + js, без сборки
- Русский язык
```

4. Деплоите на Netlify/Vercel → отправляете ссылку клиенту

**Время на заказ:** 30–60 минут с AI.

---

### Вариант B: Webhook → n8n → AI API (полуавтомат)

**Схема:** Форма → n8n → OpenAI/Claude API → генерация → email клиенту.

#### Архитектура

```
[Форма на сайте]
      ↓ POST
[n8n Webhook]
      ↓
[OpenAI GPT-4 / Claude API] — генерация HTML/CSS
      ↓
[Сохранение в GitHub / Netlify Deploy Hook]
      ↓
[Email/Telegram клиенту со ссылкой]
```

#### Настройка n8n (self-hosted или n8n.cloud)

1. Установите [n8n.cloud](https://n8n.cloud) (бесплатный trial) или self-hosted
2. Создайте workflow:

```
Webhook (POST)
  → Set (парсинг полей формы)
  → HTTP Request (OpenAI API)
  → Code (сборка HTML файла)
  → GitHub (commit файлов) ИЛИ Netlify Deploy Hook
  → Telegram (уведомление вам)
  → Email (ссылка клиенту)
```

3. Пример промпта для OpenAI node:

```json
{
  "model": "gpt-4o",
  "messages": [
    {
      "role": "system",
      "content": "Ты веб-разработчик. Создай полный HTML-сайт с встроенными CSS. Ответ — только код index.html."
    },
    {
      "role": "user",
      "content": "Создай сайт: {{$json.description}}"
    }
  ]
}
```

**Стоимость API:** ~10–50₽ за сайт при GPT-4o.

### Вариант C: Форма → Email → ручной Cursor (самый простой)

**Подходит для первых 10–20 заказов.**

1. Форма отправляет на ваш email (уже настроено через `mailto:` в `script.js`)
2. Или подключите Netlify Forms / Formspree:
   - [formspree.io](https://formspree.io) — бесплатно до 50 заявок/мес
3. Получили письмо → открыли Cursor → создали сайт → отправили клиенту

**Плюсы:** нулевые затраты, полный контроль качества.  
**Минусы:** не масштабируется.

#### Подключение Formspree

```html
<form action="https://formspree.io/f/ВАШ_ID" method="POST">
  <!-- поля формы -->
</form>
```

## 5. Приём оплаты

### ЮKassa (Яндекс)

1. Регистрация на [yookassa.ru](https://yookassa.ru)
2. Подключение как ИП или самозанятый
3. Комиссия: 3.5% + НДС
4. Интеграция: ссылка на оплату вручную или виджет на странице «Спасибо»

**Простой способ без интеграции:**
- После заявки отправляйте клиенту ссылку на оплату через ЮKassa «Счёт на оплату»

### Tinkoff Acquiring

1. [business.tinkoff.ru](https://business.tinkoff.ru) → Эквайринг
2. Комиссия от 2.5%
3. API для автоматических платежей

### Telegram Stars / перевод на карту (для старта)

- Примите оплату переводом по СБП после согласования ТЗ
- Оформите чек через «Мой налог» (самозанятый) или онлайн-кассу (ИП)

### Рекомендуемый флоу оплаты

```
1. Клиент оставляет заявку (бесплатно)
2. Вы связываетесь, уточняете ТЗ (15 мин)
3. Клиент оплачивает 1000₽ (ссылка ЮKassa / СБП)
4. AI создаёт сайт за 24ч
5. Клиент получает ссылку + 1 правка
```

## 6. Юридические основы (РФ)

> ⚠️ Обязательно перед приёмом реальных заказов!

### Что нужно оформить

| Документ | Зачем | Где взять |
|----------|-------|-----------|
| **Публичная оферта** | Договор с клиентом | Сгенерировать на [оферта.рф](https://оферта.рф) или у юриста |
| **Политика конфиденциальности** | Обработка персональных данных (152-ФЗ) | Шаблон на [consultant.ru](https://www.consultant.ru) |
| **Согласие на обработку ПД** | Чекбокс в форме | Добавить в форму заказа |

### Статус для приёма платежей

| Статус | Лимит | Налог |
|--------|-------|-------|
| **Самозанятый (НПД)** | 2.4 млн₽/год | 4–6% |
| **ИП на УСН** | без лимита | 6% |
| **Физлицо** | ❌ Нельзя для бизнеса | — |

**Рекомендация:** оформите самозанятость через приложение «Мой налог» — 10 минут, бесплатно.

### Что добавить на сайт

1. Ссылка «Публичная оферта» в footer → отдельная страница `oferta.html`
2. Ссылка «Политика конфиденциальности» → `privacy.html`
3. В форме: «Нажимая кнопку, вы соглашаетесь с политикой конфиденциальности и офертой»
4. Реквизиты в footer (ИНН самозанятого/ИП)

### Важно про маркетинг

- Отзывы на сайте должны быть реальными или помечены как «примеры»
- Фразы «осталось 3 места» — маркетинговый приём, не вводите в заблуждение в переписке
- Гарантия возврата — выполняйте, если обещаете
